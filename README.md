# Genius
> An unobtrusive AI assistant for Paper Minecraft servers

## Features
- **In-Game Assistant** - Natural language AI assistant accessible directly through Minecraft chat
- **Wide Model Support** - Compatible with any Ollama model through the Chat API
- **Per-Player Context** - Individual conversation history maintained for each player
- **Asynchronous Processing** - Non-blocking API calls ensure smooth server performance


## Installation
1. Download the latest release from [Github](https://github.com/fletchly/genius/releases/latest) or [Modrinth](https://modrinth.com/plugin/genius)
2. Place the `.jar` into your server's `plugins/` folder.
3. Restart your server
4. Follow the [quickstart guide](https://fletchly.github.io/genius-wiki/docs/quickstart) to complete setup
